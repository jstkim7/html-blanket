# HTML Blanket

Create a blank html page without typing it yourself.
Great for a quick mock-up.

To install:
```
$ npm install -g html-blanket
```
To use:
```
$ hb <filename>
```

Example:
```
$ hb index
index.html created!
```

```
index.html

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>index</title>
</head>
<body>

</body>
</html>
```
